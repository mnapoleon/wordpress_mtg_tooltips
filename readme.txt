Easily transform Arkham Horror card names into links that show the card
image in a tooltip when hovering over them. You can also quickly create deck listings.

== Description ==

This plugin will eventually show a tooltip (card image) when you hover over the card name in a blog entry.

This plugin was based on the Magic the Gathering Tooltip plugin by Sebastian Zaha and can be found at [https://github.com/SebastianZaha/wordpress_mtg_tooltips] on github.

== Installation ==

1. Head over to the "Install Plugins" section of your Admin Panel, and use the "Upload"
   link to install the zip file.
2. Activate the plugin through the 'Plugins' menu.
3. Use the "MtG" button in the editor, or manually write the tags.
